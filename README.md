# cnx-pg-notes

## 简介

`cnx-pg-notes` 是一个用于归档和发布独立 HTML 页面的仓库。  
本仓库不针对任何特定网站或域名，而是作为一个通用的 HTML 页面集合，用于保存、展示或分享各类独立文档、笔记或静态内容。

## 目录说明

```
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面引用的资源文件（如 CSS、JS、图片等）
├── README.md       # 本文件
└── index.html      （可选）仓库索引页
```

- `pages/` 目录下每个 HTML 文件代表一个独立页面，文件名建议使用有意义的英文或拼音。
- `assets/` 目录用于存放页面所需的公共资源，可按子目录组织（如 `css/`、`js/`、`img/`）。

## 页面归档说明

- 所有 HTML 页面均为独立文件，可单独访问或引用。
- 页面内容由仓库维护者自主决定，不限于特定主题或格式。
- 页面之间不强制要求链接或关联，各自独立存在。
- 页面中可能包含外部资源引用，但不保证长期可用性。

## 维护说明

- 本仓库由维护者不定期更新，新增或修改页面。
- 提交新页面时，请将 HTML 文件放入 `pages/` 目录，并确保无敏感或违规内容。
- 如需引用资源，请优先使用 `assets/` 目录下的本地文件。
- 不建议在页面中添加跟踪脚本、广告或外部动态内容。

## 使用方式

您可以直接通过 GitHub Pages 或其他静态托管服务访问 `pages/` 目录下的 HTML 文件。  
也可以将仓库克隆到本地，用浏览器直接打开 HTML 文件查看。
